## üéØ What is ScrapeBI?

ScrapeBI is a powerful, user-friendly web scraping tool that allows you to extract data from websites without writing any code. Built with Python, Selenium, and Flask, it features:

- **Visual Element Selector** - Click to select elements
- **No-Code Interface** - No programming required
- **Multiple Export Formats** - JSON, CSV, TXT
- **Modern UI** - Clean, responsive design
- **Batch Processing** - Run multiple extractions
